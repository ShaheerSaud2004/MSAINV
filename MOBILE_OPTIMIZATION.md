## 🔧 Mobile-Specific CSS

All mobile optimizations are in `client/src/index.css`:

```css
/* Touch targets */
min-height: 44px;
touch-action: manipulation;

/* Prevent zoom */
font-size: 16px;

/* Smooth scrolling */
-webkit-overflow-scrolling: touch;

/* Tap highlights */
-webkit-tap-highlight-color: rgba(59, 130, 246, 0.1);

/* Text sizing */
-webkit-text-size-adjust: 100%;
```
