## ğŸ› Troubleshooting

### Port Already in Use
```bash
# Kill process on port 3021
lsof -ti:3021 | xargs kill -9

# Kill process on port 3022
lsof -ti:3022 | xargs kill -9
```

### React Not Loading Environment Variables
Stop the React dev server (Ctrl+C) and restart:
```bash
cd client
npm start
```

### Cannot Connect to Backend
1. Ensure MongoDB is running (if using mongodb mode)
2. Check both servers are running
3. Clear browser cache
4. Check console for errors

### CORS Errors
- Make sure `CLIENT_URL=http://localhost:3021` in `server/.env`
- Make sure `REACT_APP_API_URL=http://localhost:3022/api` in `client/.env`
- Restart both servers after changing .env files
